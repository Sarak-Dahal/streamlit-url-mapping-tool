# Streamlit URL Mapping Tool

A Streamlit application for mapping URLs and managing redirects using fuzzy matching with PolyFuzz.

## Features

- Upload CSV files containing old and new URLs.
- Use fuzzy matching to map old URLs to new ones.
- Download the mapping results as a CSV file.

## Installation

1. **Clone the repository**:
   ```sh
   git clone https://github.com/sarak-dahal/streamlit-url-mapping-tool.git
   cd streamlit-url-mapping-tool

2. pip install -r requirements.txt

3. streamlit run app.py
